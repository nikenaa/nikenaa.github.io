About me

Perkenalkan nama saya Niken Ayunda Lestari, saya biasa dipanggil Niken. Saya sekarang menjadi mahasiswa Stmik Widya Pratama Pekalongan, selain itu saya seorang pekerja paruh waktu di toko hp di kota saya. Pada pagi hari saya berkuliah sampai hari sabtu lalu sorenya kerja, ketika hari minggu saya biasanya lembur dikerjaan. Saya tidak tau apa cita cita saya, yang penting bisa menghasilkan uang dengan cara halal. Kenangan saya selama setahun belakangan ini adalah saat mengikuti bios, pada saat itu jadwal bios bertabrakan dengan jadwal bekerja saya, untungnya kakak tingkat dengan baik hati memberi saya dispensasi waktu. Hobi saya adalah menonton drama korea saat senggang dan berenang.
